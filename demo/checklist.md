# Hackathon demo checklist

- [ ] Install dependencies and verify production build
- [ ] Add a fresh `OPENAI_API_KEY` in `.env.local`
- [ ] Prepare one public demo repository
- [ ] Confirm Git is available to the app process
- [ ] Rehearse the 3-minute script twice
- [ ] Record pitch video and separate usage video
- [ ] Verify no credentials appear in terminal, browser, or video
- [ ] Export one generated Markdown document during recording
