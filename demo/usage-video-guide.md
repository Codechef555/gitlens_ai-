# Separate usage video — run and use GitLens AI

## Goal

Create a 2–4 minute tutorial separate from the pitch video.

## Flow

1. Show `npm install` and copying `.env.example` to `.env.local`.
2. Explain that `OPENAI_API_KEY` stays local and is never committed.
3. Run `npm run dev`.
4. Import a public GitHub repository.
5. Open the workspace and navigate Architecture, Dependencies, Change impact, and Risks & docs.
6. Open Chat, ask a repository question, and show the streamed response.
7. End by showing Markdown export and the README.

## Recording notes

Do not record an API key. Use a redacted terminal or edit the `.env.local` step in post-production. Add chapter cards: Setup, Import, Explore, Simulate, Chat, Export.
