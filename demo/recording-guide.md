# Recording guide

Record at 1440×900 or 1920×1080, using a dark desktop theme. Start with a clean browser profile and a prepared public demo repository with recognizable API, auth, and test files. Use a 5–8 second pause after each major view change. Keep browser chrome hidden where possible, record system audio only if narration is live, and export H.264 MP4 at 1080p.

For the separate “how to run and use” video, follow `demo/usage-video-guide.md`. Keep it distinct from the judging pitch: it should show setup, `.env.local`, import, workspace navigation, and chat usage.
